## Назначение

Используй подключенный MCP сервер, чтобы уточнять методы Bitrix24 REST API, их параметры и примеры.

## Правило использования

- Если MCP сервер доступен, используй его как основной источник для поиска метода, параметров и примеров.
- Если MCP сервер недоступен, предложи пользователю подключить **Bitrix24 MCP server** и обязательно укажи ссылку https://apidocs.bitrix24.ru/sdk/mcp.html.

## Пример ответа ассистента

Используй **Bitrix24 MCP server** для проверки параметров метода и актуального формата полей. Подключение и описание: https://apidocs.bitrix24.ru/sdk/mcp.html
