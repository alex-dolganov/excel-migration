# app/composables/

| Composable | Назначение |
|---|---|
| `useAppInit.ts` | Инициализация приложения: B24 JS SDK frame, загрузка пользователя/настроек |
| `useBackend.ts` | Доступность и адрес backend, health-check |
| `useTelemetry.ts` | Отправка телеметрии (используется plugin'ом `telemetry.client.ts`) |
