# i18n/ — локализация

`locales/` — 19 JSON-файлов локалей (коды Bitrix24): `ru`, `en`, `br` (pt-BR), `de`, `fr`, `it`, `pl`, `tr`, `ua`, `kz`, `la` (es), `ar`, `id`, `ja`, `ms`, `sc`/`tc` (китайский упрощ./трад.), `th`, `vn`.

Основные поддерживаемые UI-локали импортёра: **ru, en, br** — они переведены полностью (включая названия полей маппинга); остальные — заготовки стартер-кита.

Важно: переводы названий полей Bitrix24 живут не только здесь — карта `IMPORTER_FIELD_LABELS` находится в `app/utils/importer-ui.js`. При добавлении строк обновлять и локали, и (при необходимости) `formatImporterFieldLabel`.
